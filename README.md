# doraemon

>项目中会使用到的工具,希望能可以像哆啦A梦一样的帮助到我，哇哈哈哈！

![cover](./doraemon_cover.jpg)

## 安装

```bash
$ npm install -g nd-doraemon
```

国内环境，对以上两个命令添加参数 `--registry https://registry.npm.taobao.org`，可提高安装速度

## 使用

### 1、查看帮助

```bash
$ doraemon [command] －h
```

### 2、移除模块

```bash
$ doraemon modules
```

### 3、加载所有模块

```bash
$ doraemon modules -n
```
